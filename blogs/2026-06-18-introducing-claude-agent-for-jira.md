Claude Agent for Jira, built on Anthropic's Claude Managed Agents infrastructure, enables direct work item assignment to Claude. The agent operates in a sandboxed environment, analyzes codebases, implements changes, and opens draft pull requests, offering managed cloud workflows and local terminal/IDE integration through model context protocol servers and the Teamwork Graph CLI.

## Features

| Name | Description |
|------|-------------|
| Jira Software | Agile project management and issue tracking platform for software development teams with Scrum and Kanban boards. |
| Confluence | Team workspace for creating, sharing, and collaborating on documentation, knowledge bases, and project plans. |
| Bitbucket | Git-based source code repository hosting with built-in CI/CD pipelines, code review, and branch management. |
| Jira Service Management | IT service management platform with ITSM workflows, incident management, asset tracking, and customer portals. |
| Trello | Visual project management tool using boards, lists, and cards for organizing work and tracking progress. |
| Statuspage | Status page hosting service for communicating system status, incidents, and scheduled maintenance to stakeholders. |
| Compass | Developer experience platform for tracking software component health, dependencies, and team ownership. |
| Forge Platform | Serverless cloud app development platform for building Atlassian integrations hosted on Atlassian infrastructure. |
| Atlassian Connect | Framework for building cloud apps that extend Jira, Confluence, and other Atlassian products with custom functionality. |
| Atlassian Admin | Organization-level administration for managing users, groups, domains, policies, and product access across Atlassian Cloud. |
| Atlassian MCP Server | Remote MCP server that connects Jira and Confluence with LLMs, IDEs, and AI agent platforms. |

## Use Cases

| Name | Description |
|------|-------------|
| Agile Software Development | Plan, track, and release software using Scrum and Kanban workflows with Jira Software boards and sprints. |
| DevOps and CI/CD | Automate build, test, and deployment pipelines using Bitbucket Pipelines integrated with Jira for end-to-end traceability. |
| IT Service Management | Manage IT service requests, incidents, problems, and changes using Jira Service Management with SLA tracking. |
| Knowledge Management | Create and organize team documentation, runbooks, and knowledge bases using Confluence spaces and templates. |
| Enterprise Administration | Manage organization-wide user provisioning, access policies, and product licenses across Atlassian Cloud products. |
| Incident Management | Detect, respond to, and resolve major incidents with JSM operations alerts, on-call schedules, and postmortem tracking. |
